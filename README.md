# Advanced Java 

A collection of Java programs covering important Advanced Java concepts and the Java Collection Framework.
This repository focuses on strengthening advance Java topics through hands-on implementation of modern Java features and collections.

## Topics Covered

### Advanced Java

* Enum
* Lambda Expressions
* Exception Handling

  * try-catch
  * throw & throws
  * Custom Exception
* Multithreading
* Runnable Interface
* Race Condition

### Collection Framework

* ArrayList
* Iterator
* Stack
* Queue & PriorityQueue
* ArrayDeque
* HashSet
* LinkedHashSet
* TreeSet
* HashMap
* TreeMap
* Arrays Class
* Collections Class
* Comparable & Comparator
* forEach Method
* Stream API
* Parallel Stream
* Optional Class
* Method Reference
* Constructor Reference

## Tech Stack

* Java
* VS Code
* JDK 8+

## Run the Programs

```bash
javac FileName.java
java FileName
```
